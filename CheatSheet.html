Answer

1. The most essential technologies that made the world Wide Web such a success are: HyperText Markup Language (HTML), Uniform Resource Identifier (URI) and HyperText Transfer Protocol (HTTP)

2. Resources are addressed with the URL.

3. The general components are a scheme, a host, a path and a query string.

Web Server

1. A web server refers to a server software, or hardware dedicated to running sever, that can serve contents to the World Wide Web. A web server processes incoming network requests over HTTP and several other related protocols.

2. The primary function of a web server is to store, process and deliver web pages to clients.

3. The resource is typically a real file on the server's secondary storage, but this is not necessarily the case and depends on how the web server is implemented.

4. The content is described with tags.

Web Client

1. A web browser (commonly referred to as a browser) is a software application for accessing information on the World Wide Web.

2. The display web pages, images, and videos on the user's device

3. A web client gets data and the web browser processes the data
